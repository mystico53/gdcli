# gdcli

This repo **is** gdcli — the CLI/MCP tool you may be using right now to interact with Godot projects. The developer who maintains it is the same person you're working with.

If you notice anything awkward, missing, or broken while using gdcli (via MCP tools or the CLI), mention it. Friction you hit is valuable feedback — bad defaults, unclear errors, missing features, surprising behavior, etc. Just bring it up naturally; no special format needed.
